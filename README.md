# Golang Project Setup with Fiber, MongoDB, and Firebase

This repository provides the initial setup for a Golang project using the Fiber framework, MongoDB as the database, and Firebase for authentication and other services. It includes essential configurations and middleware for quick project initialization.

## Features
Fiber Framework: Lightweight, fast web framework for building REST APIs in Go.
MongoDB: Configured database connection to handle data storage and retrieval.
Firebase: Integrated Firebase for user authentication and middleware for request validation.
